## Assignment 1 ‚Äî NLP Preprocessing Pipeline 

*This assignment focuses on transforming raw text into a clean, machine-readable format.
A custom paragraph is processed step by step using core NLP techniques.*

##### *üîπ Steps Covered*
*1. Tokenization*

*Splitting text into smaller units (words).*

*2. Stopword Removal*

*Removing common words that do not add much meaning (like ‚Äúis‚Äù, ‚Äúthe‚Äù, ‚Äúin‚Äù).*

*3. Stemming*

*Reducing words to root forms by removing prefixes/suffixes.
Example: playing ‚Üí play, studies ‚Üí studi*

*4. Lemmatization
Converting words to meaningful dictionary base forms using grammar rules.
Example: studies ‚Üí study, better ‚Üí good*

*To understand how text is cleaned and prepared before applying any NLP or ML model.*

*5. Model Evaluation*
